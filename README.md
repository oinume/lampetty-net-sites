# lampetty-net-sites

## Set up

Install NodeJS.

```
brew install node
```

OR

```
nodebrew install v12.18.3
```

Install gatsby-cli.

```
npm install -g gatsby-cli
```

Set up www as portfolio site.

```
gatsby new www https://github.com/santosfrancisco/gatsby-starter-cv
```

## www

```
cd www
npm run develop
```

Open http://localhost:8000/ with browser.
